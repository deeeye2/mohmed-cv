DevOps Bot Installation Guide

System Requirements

Ensure your system meets the following requirements:

  • Memory: 2GB RAM (4GB recommended)
  • CPU: 2 cores (4 cores recommended)
  • Disk Space: 1GB (5GB recommended)
  • OS: Ubuntu 20.04 or higher

Installation Steps

#!/bin/bash

# Variables
REPO_URL="https://github.com/deeeye2/devops-bot.git"
INSTALL_DIR="$HOME/devops_bot"

# Update and install required packages
echo "Updating system and installing required packages..."
sudo apt-get update
sudo apt-get install -y python3 python3-venv python3-pip git

# Clone the repository
if [ -d "$INSTALL_DIR" ]; then
    echo "Directory $INSTALL_DIR already exists. Please remove it before running this script."
    exit 1
fi

git clone $REPO_URL $INSTALL_DIR

# Navigate to the installation directory
cd $INSTALL_DIR

# Create a virtual environment
python3 -m venv env

# Activate the virtual environment
source env/bin/activate

# Upgrade pip and install wheel
pip install --upgrade pip
pip install wheel

# Install the devops-bot package
pip install .

Uninstallation Guide

            
#!/bin/bash

# Stop the DevOps Bot Service
sudo systemctl stop devops_bot.service

# Disable the DevOps Bot Service
sudo systemctl disable devops_bot.service

# Remove the systemd service file
sudo rm /etc/systemd/system/devops_bot.service

# Reload systemd
sudo systemctl daemon-reload

# Deactivate the virtual environment if active
deactivate

# Remove the DevOps Bot directory
rm -rf ~/devops_bot

# Remove the symbolic link
sudo rm /usr/local/bin/devops-bot

# Optionally remove installed Python packages
rm -rf ~/devops_bot/env

# Optionally remove system-wide dependencies (be cautious)
sudo apt-get remove --purge -y python3 python3-venv python3-pip git
sudo apt-get autoremove -y

echo "DevOps Bot uninstalled successfully."
